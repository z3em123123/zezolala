# Z3em Portfolio

Portfolio website for Z3em - Full Stack Developer

## Getting Started

```bash
yarn install
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Built With

- Next.js 14
- React 18
- Tailwind CSS
- GSAP Animations
